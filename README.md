# Student Management System

## Project Description

The Student Management System is a Java-based application developed to manage student records.
This project performs CRUD operations (Create, Read, Update, Delete) using Java, JDBC, and MySQL database.

## Technologies Used

* Java
* JDBC
* MySQL
* SQL
* Eclipse / IntelliJ IDE
* Git & GitHub

## Features

* Add new student details
* View student records
* Update student information
* Delete student records from database
* Connect Java application with MySQL using JDBC

## Database

Database Name: student_management

Table Name: student

Fields:

* id
* name
* age
* course
* email

## How to Run the Project

1. Install MySQL and create the database.
2. Create the student table in MySQL.
3. Open the project in Eclipse or IntelliJ.
4. Update database username and password in the JDBC connection.
5. Run the Java application.
